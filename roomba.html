ROOMBA INFORMATION

The name of the Roomba is Dilir. This name came to be because Adam and Ethan were in math class. One of their assignments included a process called D.I.L.I.R the robot and it just kinda clicked with us. We control the Roomba using a Python program that we run on a Raspberry Pi which we have connected and controlled from another Laptop using the program "VNC Viewer" similar to Team Viewer

ROOMBA CONTROLS

P - Passive

S - Safe

F - Full

C - Clean

D - Dock

R - Reset

Space - Beep

Arrows - Motion
